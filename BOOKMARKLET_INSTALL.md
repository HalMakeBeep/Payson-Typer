## Install it!

1. Open [BOOKMARKLET_INSTALL.html](https://halmakebeep.github.io/Payson-Typer/BOOKMARKLET_INSTALL.html).
2. Drag **Payson Typer (Drag Me)** to your bookmarks bar.

## If drag does not work or your super retarded:

1. Open `paysontyper.bookmarklet.js`.
2. Copy all text.
3. Create a bookmark manually.
4. Paste the copied text into the bookmark URL field.
